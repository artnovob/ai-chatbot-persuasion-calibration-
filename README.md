# AI Persuasion Calibration Chat Interface

A minimalist chat interface for research on "Calibrating Persuasion in AI Commerce Agents". 

## Live Demo
[Click here to use the chat](https://ai-persuasion-calibration-chat.vercel.app)

## Features
- Clean chat interface
- Hidden Emotional/Rational mode toggle
- Groq API integration
- Local browser storage for API keys

## How to Use
1. Open `science_chatbot.html` in browser
2. Enter Groq API key (get from console.groq.com)
3. Start chatting

## Research
Study of communication styles (emotional vs. rational) in AI-human interactions.
